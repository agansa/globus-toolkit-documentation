Technology Preview-5

  • New Service Data, Notification, and Query Engine API
  • Service Data Annotation Support using Doclets
  • New Routing and Virtual Hosting Environment Framework
  • Service Data Element declarations in WSDL
  • New Open Source based GSI implementation
  • X509 based XML Signature support
  • TypeMapping and Extensibility Element Simplifications
  • Chained WSDL Exceptions
  • Packaging and Deployment improvements

Technology Preview-4

  • Implementation of GGF Grid Services Specification Draft 4 [10/03/2002]
  • New packaging model for 3rd party service deployment
  • New documentation: Programmers Guide, and White Papers
  • Initial C client support
  • Instrumentation support
  • Grid Service Locator Generator
  • J2EE JMS - OGSI Notification Gateway
  • Lifecycle monitor interceptors and automated deactivation

Technology Preview-3

  • Implementation of GGF Grid Services Specification Draft 3 [07/17/2002]
  • Index Service Prototype
  • Reliable File Transfer Service Prototype
  • WS-Security based GSI over SOAP implementation using W3C XML Signature and Encryption and Java GSSAPI over PKI
  • EJB Dispatcher Generator - Support for exposing EJB Session and Entity Beans as Grid services
  • Support for Service Activation/Deactivation
  • Support for persistent service properties
  • New soft-state and factory based notification infrastructure
  • ServiceDataDescription (SDD) Support in WSDL
  • GS Spec Draft 3 Service Data compatibility tests
  • Upgrade to Axis beta 3, and Ant 1.5

Technology Preview-2.1

  • Improved build environment
  • Improved user's guide
  • Improved Service Browser GUI
  • rpc/encoded + doc/literal mix support
  • Java 2 WSDL generation support
  • Google integration sample
  • .NET Visual Studio sample
  • Web Start Support

Technology Preview-2

  • document/literal WSDL
  • .NET Integration (client side)

  • GSH/GSR mapping and inspection
