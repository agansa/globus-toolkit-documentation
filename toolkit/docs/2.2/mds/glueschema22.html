MDS 2.2 GLUE Schemas

Please go here for the latest information on GLUE schema.

MDS 2.2 includes initial support for Grid Laboratory Uniform Environment (GLUE) requirements by providing the GLUE schema (in $GLOBUS_LOCATION/etc/grid-info-glue.schema) described in this document.

MDS 2.2 also provides a single host, single node information provider. The grid-info-resource-ldif.conf file has an entry for this provider, which can provide MDS information in GLUE format to a GRIS, and the grid-info-search command can locate and output the GLUE formatted information. This provider entry is described in MDS 2.2 Configuration Files. The use of the provider is described in MDS 2.2 New Features.

Note that the use of the GLUE schema is optional. Existing MDS 2.1 users can continue to use their current providers and schema if they wish.

Core MDS GRIS provider schema are documented separately in MDS 2.2 Schemas.

The GLUE schema is as follows:
