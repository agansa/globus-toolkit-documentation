Data management tools are concerned with the location, transfer, and management of distributed data, as described in Data Management: Key Concepts.

GT4's primary data management tools are:

  • GridFTP [pdf] for high-performance and reliable data transport
  • Reliable File Transfer (RFT) service [pdf] for managing multiple transfers
  • Replica Location Service (RLS) [pdf] for maintaining location information for replicated files
  • Data Replication Service (DRS) [pdf]
  • OGSA Data Access and Integration (OGSA-DAI) [pdf] for accessing and integrated structured and semistructured data
