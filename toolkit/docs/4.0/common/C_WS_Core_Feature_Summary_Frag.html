C WS Core Feature Summary

New Features in the GT 4.0 release

  • Implementation of the OASIS WSRF and WSN working drafts specifications (with minor fixes to the 1.2-draft-01 published schemas and with March 2004 version of WS-Addressing specification)
  • HTTP/1.1 client & server support
  • Generation of pure C stubs (including blocking and asynchronous versions) directly from WSDL schemas
  • Dynamically loadable Operation Providers and Service Modules based on the new Extension API included in the Globus C Common libraries
  • API for embedding services in a C application
  • Stand-Alone service container
  • Resource API for managing resources from within a service
  • Use of the Libxml2 Pull Parser for efficient messaging parsing
  • Use of XIO HTTP driver
  • full UUID support
