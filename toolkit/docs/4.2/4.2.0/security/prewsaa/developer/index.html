Introduction

This component provides two main APIs for authentication and authorization. The main authentication API is a implementation of the GSSAPI RFC extended with functions described in the GSSAPI Extensions [add links] document. The main authorization API is captured in a single function: globus_gss_assist_map_and_authorize(...).
