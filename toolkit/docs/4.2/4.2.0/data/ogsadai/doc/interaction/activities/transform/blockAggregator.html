Activity - blockAggregator

Activity Type: Transformation

Description

Aggregate the blocks of data output from other activities into a single, larger block. This can be used to reduce network traffic when transferring data, for example.

Example

<blockAggregator name="aggregator">
  <input from="blocksToAggregate"/>
  <blocksize size="10"/>
  <output name="aggregatedBlocks"/>
</blockAggregator>

Specification

Element blockAggregator:

  • Attribute name - unique name for the activity.
  • Element input (required) - the string data blocks to aggregate.
    • Attribute from - input stream providing the blocks.
  • blocksize (required) - aggregation size.
    • Attribute size - number of blocks to be aggregated.
  • output (required) - output stream for the aggregated block.
    • Attribute name - name of this output stream.

Limitations

Currently only string blocks can be aggregated.
