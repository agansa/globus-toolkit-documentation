DeliverToStreamServlet

The DeliverToStreamServlet is a servlet which can be deployed within a Web services container and receives outputs from the deliverToStream activity.

The servlet must be installed in the same Web services container as OGSA-DAI.

Installation

To install the DeliverToStreamServlet:

  1. Edit the file CONTAINER/webapps/ogsa/WEB-INF/web.xml:
    • Add the following <servlet> description at the bottom of the list of <servlet> descriptions: 
      <servlet>
  <servlet-name>DeliverToStreamServlet</servlet-name>
  <display-name>OGSA-DAI Deliver To Stream Servlet</display-name>
  <servlet-class>
    uk.org.ogsadai.activity.delivery.DeliverToStreamServlet
  </servlet-class>
</servlet>
    • Add the following <servlet-mapping> description at the bottom of the list of <servlet-mapping> descriptions: 
      <servlet-mapping>
  <servlet-name>DeliverToStreamServlet</servlet-name>
  <url-pattern>servlet/DeliverToStreamServlet</url-pattern>
</servlet-mapping>
  2. Shutdown and restart your Web services container.

Usage

When the deliverToStream activity is executed then data will be streamed to the following URL: 

http://HOST:PORT/ogsa/servlet/DeliverToStreamServlet?url=
  http://HOST:PORT/ogsa/ogsadai/GridDataServiceFactory/hash1234567890
  &streamId=NAME

where NAME is the value of the name attribute of the deliverToStream activity.
