Activity - fileQuery

Activity Type: BinX

Description

Run an XPath query over a Binary XML (BinX) file.

Examples

<fileQuery name="fileQuery">
  <queryPath>
    /some/directory/some/file.bin//*[@varName='params']
  </queryPath>
  <output name="myOutput"/>
</fileQuery>

Specification

Element fileQuery:

  • Attribute name - unique name for the activity.
  • Element queryPath (required) - contains an XPath query prefixed by a path to a file, e.g. the example above runs the XPath query //*[@varName='params'] on the BinX file at /some/directory/some/file.bin.
  • output (required) - output stream for the transformed data (characters).
    • Attribute name - name of this output stream.
