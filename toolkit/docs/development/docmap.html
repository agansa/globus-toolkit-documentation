Status and Plans for GT4 Documentation:
A map of the docs in support of GT4 Final

Draft: 9 September 2004

This document is an attempt to describe the body of documentation planned in support of the GT 4.0 release by the Globus Alliance. The purpose of making this public is twofold:

  1. To invite comments on our plans
  2. To expose opportunities for contributing to the GT document effort

The document map is comprised of the following sections, each of which covers different areas of the Globus Toolkit:

  • General GT docs
  • Security docs
  • Data Management docs
  • Execution Management docs
  • Information Services docs
  • Common Runtime Component docs
The feedback mechanism for these documents is the Globus Bugzilla System. If you wish to submit a comment or request an improvement you must create a new bug entry. Most documents in the map have a template associated with them. A single template defines the structure for multiple documents. Changes to templates imply changes in all derived docs, and so can carry with them a great deal of work. Please suggest changes to templates sooner, rather than later!

We are using the Globus Bugzilla System to track comments and change requests regarding GT4 docs. To log your comments and change requests for improving the docmap, please create a new bug.
