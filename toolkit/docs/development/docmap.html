Status and Plans for GT4 Documentation:
A map of the docs in support of GT4 Final

Draft: 13 September 2004

This document is an attempt to describe the body of documentation planned in support of the GT 4.0 release by the Globus Alliance. The purpose of making this public is twofold:

  1. To invite comments on our plans
  2. To expose opportunities for contributing to the GT document effort

The document map is comprised of the following sections, each of which covers different areas of the Globus Toolkit:

  • General GT docs
  • Security docs
  • Data Management docs
  • Execution Management docs
  • Information Services docs
  • Common Runtime Component docs

How you can participate

The feedback mechanism for documentation is based on the Globus Bugzilla System. If you wish to offer a suggestion regarding our documentation (or contribute content!) please create a new bugzilla entry in the appropriate area. Please ensure that all documentation entries have a Product name of "Documentation" and be assigned one of the following Components: General, Security, Data Management, Information Services or Common Runtime Components. Feedback regarding the docmap itself can be filed under Documentation/General.

Please note that most documents in the map have a template associated with them. A single template can define the structure for multiple documents, and so changes to them can imply great deal of work. Because of this, early feedback on document templates is appreciated.
