Status and Plans for GT4 Documentation:
A map of the docs in support of GT4 Final

Draft: 7 August 2004

This document is an attempt to describe the body of documents planned in support of the 4.0 release by the Globus Alliance. The purpose of making this public is twofold:

  1. To invite comments on our plans
  2. To expose opportunities for contributing to the GT document effort
Most document types in the map have a template associated with it. The template defines the overall structure for that class of document. Changes to templates imply changes in all associated docs, and so can carry with them a great deal of work. Please suggest changes to templates sooner, rather than later!

The map is broken out into the following sections:

  • General GT docs
  • Security docs
  • Data Management docs
  • Resource Management docs
  • Information Services docs
  • Deployment Environment docs
