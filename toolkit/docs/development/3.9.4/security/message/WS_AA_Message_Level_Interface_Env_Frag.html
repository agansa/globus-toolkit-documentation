• X509_USER_PROXY <path>, where <path> is the path of the proxy credential
  • X509_USER_KEY <path>, where <path> is the path to a private key. Requires that the corresponding certificate is set with X509_USER_CERT
  • X509_USER_CERT <path>, where <path> is the path to a certificate. Requires that the corresponding private key is set with X509_USER_KEY
  • X509_CERT_DIR <path>, where <path> is the path to the trusted certificates directory, i.e. the directory containing the trusted CA certificates

Note that the above environment variables do not supersede any settings provided in security descriptors.
