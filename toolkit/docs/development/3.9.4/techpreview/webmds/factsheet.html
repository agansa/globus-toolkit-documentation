WebMDS factsheet for GT 4.0 tech preview

Brief Component Overview

WebMDS allows monitoring information to be presented to end users through a standard web browser interface. This means that end users do not need to install any software on their PC. WebMDS is a redesign and reimplementation of a student project, WebSDB, which was released through the GTR and gave a similar interface for GT3 MDS service data.

Summary of features

  • Collects monitoring information from pluggable sources:
    • By default, by using WSRF Resource Properties poll operations.
    • MDS archiver plug in development to display historical information
  • Formatting of monitoring data (in XML) into HTML output is performed using pluggable XSLT style sheets. The 4.0 release contains a small number of unattractive stylesheets as examples, with it intended that subsequent releases will contain substantially improved (more useful and more visually appealing) stylesheets.
