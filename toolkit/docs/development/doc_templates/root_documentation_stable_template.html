Draft: 2 September 2004
Target readers: Non-experts, Developers, Sysadmins, End-users

GT release documentation is organized into the following manuals:

Key Concepts

Provides a high-level overview and discussion of fundamental concepts

Developer's Guide

Describes how to use GT programming interfaces

System Administrator's Guide

Describes information needed by people who install, deploy and maintain GT installations

User's Guide

Describes how to use the human-driven interfaces (command-line tools, GUIs, etc.) distributed with GT
