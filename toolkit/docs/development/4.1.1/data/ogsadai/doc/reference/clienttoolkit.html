Client Toolkit Examples

This table below contains links to each of the examples covered in the Client Toolkit Tutorial. For more details please refer to the tutorial. For comprehensive documentation of the Client Toolkit APIs please refer to the OGSA-DAI Javadoc.

Example Description
The Basics
Locating A Factory Locates and connects to a Grid Data Service Factory with a known handle.
Create and Destroy GDS Creates a Grid Data Service using a Grid Data Service Factory then destroys the Grid Data Service again.
Performing Queries
Simple SQL Query Creates a Grid Data Service and performs a simple SQL query. The results are returned within the response document and then displayed on the console.
Sequence of SQL Queries Constructs a request containing several queries.
Parameterised SQL Query Parameterises an SQL query then runs it repeatedly with different parameter values.
Simple XPath Query Performs a simple XPath query against an XML collection. The results are returned within the response document and displayed on the console.
Processing Results
Processing SQL Query Results Processes the results of an SQL query using the JDBC ResultSet interface.
Processing Larger SQL Query Result Sets Process the results of an SQL query that returns a large number of rows. The rows are returned asynchronously from the server using an OutputStreamActivity, so that the entire results do not have to be collated on the server to be returned in a single response.
Processing XPath Query Results Processes the results of an XPath query using the XML:DB ResourceSet interface.
Updates and Bulkload
SQL Update Statement Performs an update to the OGSA-DAI example database. A new row is inserted, updated and then deleted.
SQL Bulk Load Bulk loads the results of a query performed against one table into another table.
SQL Update Parameterised Via GDT Performs a parameterised update to the OGSA-DAI example database, supplying the parameters via Grid Data Transport
XUpdate Query Performs an XUpdate statement on the OGSA-DAI test database.
Accessing a Registry
Listing Registered Factories Queries a registry and lists the handles and implemented port types for all registered Grid Data Service Factories.
Transforming Data
Simple XSL Transform Transforms the results of an SQL query into an HTML table. In this example the XSL-Transform document is embedded in the request.
XSL Transform With Delivery Transforms the results of an SQL query into an HTML table. In this example the XSL-Transform document is loaded from a URL by the GDS.
Asynchronous Delivery
GDT Example Shows how the GDT port type can be used to push data from one Grid Data Service to another.
Data Integration
Data Integration Example A simple data integration scenario. An XPath query is performed on an XML database, the results are transformed into a WebRowSet and bulk loaded into a temporary table on a relational database. A join query is then performed and the results displayed. Finally the temporary table is removed.
