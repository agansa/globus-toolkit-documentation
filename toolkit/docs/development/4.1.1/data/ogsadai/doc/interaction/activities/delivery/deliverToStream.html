Activity - deliverToStream

Activity Type: Delivery

Description

Deliver data to the DeliverToStreamServlet to allow data to be pulled via http or https.

The data will be made available at a URL of form: 

http://host:port/ogsa/servlet/DeliverToStreamServlet?gsh=
  http://host:port/ogsa/ogsadai/DataService&streamId=NAME

where NAME is the value of the name attribute of the deliverToStream activity.

Example

<deliverToStream name="delivery">
  <fromLocal from="dataSource"/>
</deliverToStream>

Specification

Element deliverToStream:

  • Attribute name - unique name for the activity.
  • fromLocal (required) - input stream.
    • Attribute from - name of an input stream providing the data to deliver.

XML Schema

OGSA-DAI/schema/ogsadai/xsd/activities/deliver_to_stream.xsd
