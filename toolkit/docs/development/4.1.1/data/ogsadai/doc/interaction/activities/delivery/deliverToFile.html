Activity - deliverToFile

Activity Type: Delivery

Description

Deliver data to a file on the container's local filesystem.

Example

<deliverToFile name="delivery">
  <fromLocal from="dataSource"/>
  <toFile>C:/temp/file.txt</toFile>
</deliverToFile>

Specification

Element deliverToFile:

  • Attribute name - unique name for the activity.
  • fromLocal (required) - input stream.
    • Attribute from - name of an input stream providing the data to deliver.
  • Element toFile (required) - the file on the container's local filesystem to which the data is to be delivered

Warnings to Service Deployers

This activity can be used by clients to potentially overwrite any file on your server. This activity is provided for developing and testing OGSA-DAI applications.

If you intend to use files as data resources then you should support access and manipulation of these only via the file activities.
