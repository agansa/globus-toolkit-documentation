Activity - dataStore

Activity Type: Transformation

Description

Cache, buffer or provide data for other activities.

Examples

<dataStore name="parameters">
  <item>100</item>
  <item>200</item>
  <itemCursor name="parametersOutput"/>
</dataStore>

<dataStore name="parameters">
  <input from="someInput"/>
  <itemCursor name="parametersOutput"/>
</dataStore>

Specification

Element dataStore:

  • Attribute name - unique name for the activity.
  • One of the following types of element specifying the data to be cached:
    • Element item (one or more) - ontains cached data e.g. the first example above.
    • input (required) - data to be cached.
      • Attribute from - input stream providing the data to be cached e.g. the second example above.
  • itemCursor (required) - output stream for the cached data.
    • Attribute name - name of this output stream.
