Activity - relationalResourceManagement

Activity Type: Relational

Description

Run relational database management operations. The following operations are supported:

  • Create new databases.
  • Drop existing databases.

Examples

<relationalResourceManagement name="statement">
  <createDatabase>
    someDatabase
  </createDatabase>
  <resultStream name="results"/>
</relationalResourceManagement >

<relationalResourceManagement name="statement">
  <dropDatabase>
    someDatabase
  </dropDatabase>
  <resultStream name="results"/>
</relationalResourceManagement >

Specification

Element relationalResourceManagement:

  • Attribute name - unique name for the activity.
  • One of two operation elements:
    • Element createDatabase - create a new database.
      • Contains the name for new database, unique within the current database.
      • OR
      • Attribute from - input stream providing the name of the database.
    • Element dropDatabase - drop an existing database. Contains the name of the database to be dropped.
      • Contains the name for new database, unique within the current database.
      • OR
      • Attribute from - input stream providing the name of the database.
  • resultStream (required) - output stream.
    • Attribute name - name of the activity's output stream.
