Command-line tool mds-servicegroup-add for WS MDS Aggregator

Tool description

mds-servicegroup-add creates and maintains registrations to a WS-ServiceGroup. It is intended primarily for use registering grid resources to MDS services such as the index and trigger services.

Registrations are defined in an XML configuration file, which is documented in the aggregator admin guide.

Command syntax

mds-servicegroup-register /path/to/reg.xml

/path/to/reg.xml should be the full path to the configuration file.

Limitations

It is necessary for the tool to continue to run in order for the registrations that it maintains to be kept alive, as registrations will otherwise time out.
