Data management tools are concerned with the location, transfer, and management of distributed data, as described in Data Management: Key Concepts.

GT4's primary data management tools are:

  • GridFTP for high-performance and reliable data transport
  • Reliable File Transfer (RFT) service for managing multiple transfers
  • Replica Location Service(RLS) for maintaining location information for replicated files

In addition, GT4 provides tech preview versions of OGSA Data Access and Integration (OGSA-DAI) for accessing and integrated structured and semistructured data, and a Data Replication Service (DRS).
