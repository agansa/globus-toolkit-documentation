GT 3.9.2 Development Release Notes: Pre-WS GRAM

The Grid Resource Allocation Management (GRAM) service provides a single interface for requesting and using remote system resources for the execution of "jobs". The most common use of GRAM is remote job submission and control. It is designed to provide a uniform, flexible interface to job scheduling systems. Pre-WS GRAM is an implementation of GRAM based on a proprietary, pre-Web service protocol (also referred to as GRAM2).

This release contains only minor bug fixes from GT 3.2.1

For the latest documentation for Pre-WS GRAM, go to:
