GT 3.9.2 Development Release Notes: WS GRAM

The Grid Resource Allocation Management (GRAM) service provides a single interface for requesting and using remote system resources for the execution of "jobs". The most common use of GRAM is remote job submission and control. It is designed to provide a uniform, flexible interface to job scheduling systems.
