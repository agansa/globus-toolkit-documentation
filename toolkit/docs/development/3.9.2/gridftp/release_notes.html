GT 3.9.2 Development Release Notes: GridFTP

GridFTP is a protocol for secure, fast, efficient, and robust data transport. GT4 provides an implementation of this protocol. We provide a server (globus-gridftp-server), a client (globus-url-copy), and a set of libraries for developing custom applications.

In this release, GridFTP includes the following new features:

  • Supports remote data-node and striped operation.
  • globus-url-copy: Added support for transfers involving gridftp striped servers with the '-stripe' option.
