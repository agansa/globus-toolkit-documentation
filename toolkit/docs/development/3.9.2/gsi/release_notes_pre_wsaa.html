Globus Toolkit 3.9.2 Development Release Notes: Pre-WS Authentication & Authorization

This component contains libraries and tools supporting authentication and authorization for GT Pre-WS components. This includes support for various authorization modes, such as host, self, identity, gridmap and custom, as well as a SSL-based GSS-API implementation for authentication and delegation of rights.

This distribution includes no new features or API changes in the security libraries that comprise this component. However, a number of bug fixes can be found in this release.
