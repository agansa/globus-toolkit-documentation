Globus Toolkit 3.9.2 Development Release Notes: Credential Management

This component contains tools and services related to managing and creating credentials; it currently consists of:

  • SimpleCA: a set of tools designed to allow users to easily create a new certificate authority for use with GT.
  • MyProxy: an online credential repository (contributed by NCSA)

This release of the Globus Toolkit® contains SimpleCA; no new features have been added for this release.

Note: MyProxy is not included in this release; it is planned for the full-featured release in October.
