GT 3.9.2 Development Release Notes: RFT

The Reliable File Transfer (RFT) service fulfills a role similar to the batch scheduler on a compute machine. You submit a data transfer "job", which can consist of 10's or even 100's of thousands of files, along with information on buffer sizes, streams, etc. The request, along with information required for restart and notifications, is stored in a database. The service then executes a GridFTP 3rd party transfer on your behalf.

A new WSRF-compliant Reliable File Transfer Service is included in this release which can perform reliable gridftp transfers and recursive directory transfers.
