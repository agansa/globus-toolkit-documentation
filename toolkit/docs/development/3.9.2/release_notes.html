GT 3.9.2 Development Release Notes

The 3.9.2 development distribution is an alpha-quality release featuring a first look at the WSRF port of many of the WS Components. Please note that features will be added and public interfaces will change before the release of 4.0 Final. For more information on the delivery schedule and stability of interfaces, please visit the "GT4 Fact Sheet" at http://www-unix.globus.org/toolkit/GT4Facts/.

This release contains:

Security

WS Authentication & Authorization

Note: The following WS A&A features are not included in this release; they are planned for the full-featured release in October:

  • CAS
  • Generic per-method authorization framework (including support for XACML)
  • A new delegation service that decouples delegation from GSI-SecureConversation

Pre-WS Authentication & Authorization

Credential Management

SimpleCA

Note: The following Credential Management feature is not included in this release; it is planned for the full-featured release in October:

  • MyProxy (contribution)

Data Management

RFT

A new WSRF-compliant Reliable File Transfer Service is included in this release which can perform reliable gridftp transfers and recursive directory transfers.

GridFTP

RLS

XIO

This release includes the following features for XIO:

  • XIO drivers can now be dynamically loaded.
  • XIO automatically resets the EOF state to allow for 'tail' type use.
  • new GridFTP client driver.
  • new mode e driver.
  • new ordering driver.

Note: The following Data Management feature is not included in this release; it is planned for the full-featured release in October:

  • OGSA-DAI Tech Preview

Resource Management

WS GRAM

Pre-WS Gram

Note: The following Resource Management feature is not included in this release; it is planned for the full-featured release in October:

  • Community Scheduler Framework (contribution)

Information Services

MDS4

MDS2

Web Services Core

Python WS Core (contributed)

C WS Core

Java WS Core

This release of the Globus Toolkit® contains a preview implementation of the WS-ResourceLifetime, WS-ResourceProperties, WS-BaseFaults, and WS-BaseNotification specifications. This preview will evolve into the foundation of the web services based portion of the upcoming 4.0 release.
