Prerequisites

  • Java JDK v1.3 or higher

    When using IBM's JDK v1.4 you will need to obtain the Xerces-2 jars and place them in $JAVA_HOME/jre/lib/endorsed

    When using Sun's J2SE v1.4.2_05 SDK you will need to create $GLOBUS_LOCATION/endorsed directory and copy $GLOBUS_LOCATION/lib/xalan.jar file into that directory.

    Refer to other Java installation requisites in the security document.

  • Apache Ant 1.5 or higher

  • JUnit jar (allows you to run the tests.)

Installation

To install the distribution, simply run:

% install-wsrf /path/to/install

Troubleshooting Your Installation

Please ignore compilation time WARNINGS. There will be two kinds of warnings you will see that are not considered errors: 

WARNING: "/sandbox/bacon/withjunit/sandbox/bacon/withjunit/lib/libCounterService_client_bindings_gcc32dbg_gcc32dbg.a
and 
CounterPortType.c:5712: warning: value computed is not used
