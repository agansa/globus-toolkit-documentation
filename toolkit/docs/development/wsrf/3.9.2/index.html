Globus Toolkit 3.9.2 Documentation

This 3.9.2 development release is [an early preview of the Java WSRF Core implementation] that is scheduled for official release in 4.0 Final. The distribution is a full GT release and is intended for users interested in an early look at the WSRF implementation. It contains the full set of GT components with all base services (Security/GSI, GRAM, etc) ported to WSRF.

Please note that these documents will not be in their final form until the release of GT 4.0 Final.
