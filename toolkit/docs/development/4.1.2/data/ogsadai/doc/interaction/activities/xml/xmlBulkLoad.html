Activity - xmlBulkLoad

Activity Type: XML

Description

Bulk load a set of resources into an XMLDB data resource. The collection can be created before importing the documents.

Examples

<xmlBulkLoad name="bulkload">
  <collection>myCollection</collection>
  <resourceSet>
     <resource>
        <entry id="1" xmlns:src="http://xml.apache.org/xindice/Query" src:col="/db/littleblackbook" src:key="1">
           <name>Ally Antonioletti</name>
           <address>826 Hume Crescent, Southampton</address>
           <phone>01670061244</phone>
        </entry>
     </resource>
     <resource>
        <entry id="10" xmlns:src="http://xml.apache.org/xindice/Query" src:col="/db/littleblackbook" src:key="10">
           <name>Martin Laws</name>
           <address>224 Palansuriya Lane, Winchester</address>
           <phone>06686142062</phone>
        </entry>
     </resource>
  </resourceSet>
  <output name="results"/>
</xmlBulkLoad>

<xmlBulkLoad name="bulkload">
   <collection createIfNotExists="true">aNewCollection</collection>
   <fromLocal from="data"/>
   <output name="results"/>
</xmlBulkLoad>

<xmlBulkLoad name="bulkload">
   <fromLocal from="data"/>
   <output name="results"/>
</xmlBulkLoad>

Specification

Element xmlBulkLoad:

  • Attribute name - unique name for the activity.
  • Element collection (zero or one) - contains the name of the collection that the data will be loaded into. Relative to the data resource's base collection. If this element is omitted then the bulk load will be performed on the base collection for the data resource.
    • Attribute createIfNotExists (optional) false or true - specifies if the collection should be created if it does not exist in the target data resource. The default is false.
  • Element resourceSet (zero or one) - a set of resources that will be imported into the data resource. Ignored if the fromLocal element (see below) is defined.
    • Element resource (zero or more) - an XML resource that contains the document to be added to the data resource
  • Element fromLocal (zero or one) - refers to the output stream of another activity that delivers a stream of resource elements containing the bulkload data. If this element is present then the resourceSet element above will be ignored.
    • Attribute from (required) - the name of an output stream of another activity within the same perform document.
  • Element outputStream (required) - output stream.
    • Attribute name - name of the activity's output stream.
