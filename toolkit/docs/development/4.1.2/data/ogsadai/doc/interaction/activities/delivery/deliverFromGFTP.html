Activity - deliverFromGFTP

Activity Type: Delivery

Description

Retrieve data from a GridFTP server.

Example

<deliverFromGFTP name="retrieve">
  <fromGFTP host="somehost.org" port="8080" file="path/file.ext"/>
  <toLocal name="dataSink"/>
</deliverFromGDT>

Specification

Element deliverFromGFTP:

  • Attribute name - unique name for the activity.
  • Element fromGFTP (required) - the GridFTP server from which the data is to be retrieved.
    • Attribute host (required) - GridFTP server host.
    • Attribute port (required) - GridFTP server port.
    • Attribute file (required) - GridFTP server path and file.
  • toLocal (required) - output stream.
    • Attribute name - name of this activity's output stream.
