Globus Standard

  • GLOBUS_LOCATION <path> - where <path> is the root location of the Java WS Core installation. (Required)
  • X509_USER_PROXY <path> - where <path> is the path to the proxy credentials. It is used to specify a specific credential to use instead of using the default user credentials. (Optional)
  • GLOBUS_TCP_PORT_RANGE <min,max> - where <min,max> is the minimum and maximum port range for TCP server sockets (usefull for systems behind firewalls). For example, if set, the notification sink on the client will be started within that port range. (Optional)
  • GLOBUS_HOSTNAME <host> - where <host> is either a hostname or ip address. The host ip address under which the container and services will be exposed. (Optional)

Launch script specific

  • GLOBUS_OPTIONS <arguments> - where <arguments> are arbitrary arguments that can be passed to the JVM. (Optional)
  • JAVA_HOME <path> - where <path> is the root location of the JVM installation. If set, JVM from that installation will be used. Otherwise, the fist one found in path will be used. (Optional)
  • CLASSPATH <classpath> - this environment property is ignored
