GT 4.0 System Administrator Guides

The first link, GT 4.0 System Administrator's Guide, is the starting point for everyone who wants to install Globus Toolkit 4.0. It will take you through a basic installation that installs Java WS Core and the following Base Services: a security infrastructure (GSI), the index service, and GridFTP. Component-level system administrator's guides provide any further installation, configuration and deployment information specific to that component.

Important: We highly recommend that you fully read through the toolkit-level system administrator's guide, then the system administrator's guide of the component of your choice before attempting installation.
