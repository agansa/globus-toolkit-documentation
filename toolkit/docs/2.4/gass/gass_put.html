Put a file to a remote server

Synopsis

globus-gass-put Filename URL

Description

The globus_gass_put utility allows a user to upload a file using the GASS client library. This program reads the file from the locally accessible filesystem, but does not use the gass cache. To access locally cached URLs, use the globus_gass_cache utility.

Options

Filename

the local file name

URL

the remote URL to send the file to. It can be of any URL scheme supported by GASS (currently x-gass and ftp).

Return Values

Upon successful completion, the globus_gass_put program will return an exit value of 0, and toURL will be replaced with the contents of fromFile.

Examples

globus-gass-put ~/file.dat ftp://gass.globus.org/pub/incoming/file.dat

globus-gass-put ~/file.dat x-gass://gass.globus.org:1440/~/file.dat
