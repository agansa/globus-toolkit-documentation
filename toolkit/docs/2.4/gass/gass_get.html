Fetch a remote URL

Synopsis

globus-gass-get URL filename

Description

The globus-gass-get utility allows a user to download a file using the GASS client library. This program places the file in the locally accessible filesystem, but does not use the gass cache. To access locally cached URLs, use the globus-gass-cache utility.

Options

URL

the remote file to fetch. It can be of any URL scheme supported by GASS (currently x-gass and ftp).

Filename

the local file name that you wish the file to be fetched to. It can be either a local directory or a local filename. If it is a directory, then the final path specification of the URL is used.

Return Values

Upon successful completion, the globus_gass_get program will return an exit value of 0, and the toFile will be populated with the contents of fromURL.

Examples

globus-gass-get ftp://gass.globus.org/pub/file.dat /tmp/file.dat

globus-gass-get x-gass://gass.globus.org:1440/~/file.dat file2.dat
