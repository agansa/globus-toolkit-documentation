Notes:

    I have successfully installed the Globus Toolkit on HP-UX. But I had to apply a few patches to the SASL and OpenLDAP configuration files.

    Also some of the update releases breaks HP-UX support due to an old version of ltconfig. (The original version had a newer one.)

    You can find the patches on http://grid.tsl.uu.se/hpux/ which describes the installation procedure I used.

    The installation script (downloadable from that page) also does some clean-up, like removing recursive symbolic links (the link points to itself), setting some file permissions and cleaning the OpenLDAP .la files from build directory paths.

    There are also some scripts I wrote to fill the information in the MDS for HP-UX.

    I hope these patches and clean-ups can be integrated in the official distribution.
