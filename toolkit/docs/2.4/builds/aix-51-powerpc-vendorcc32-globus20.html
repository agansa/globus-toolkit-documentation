Notes:

    DATA MANAGEMENT SERVER BUNDLE:
    To build the globus data management server bundle, you must first set the environment variable CC to be "cc".  That will force compiler selection to be cc.  Then follow the normal build instructions.

    INFORMATION SERVICES SERVER BUNDLE:
    There are known building issues that prevent this bundle from working correctly.  We have identified a work around for this, which will be posted soon.  The AIX information providers have been supplied by IBM, and will soon be available for download.
