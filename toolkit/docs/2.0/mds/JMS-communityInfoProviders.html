INFORMATION PROVIDERS BEING DEVELOPED

  • Software Information Provider

http://www.thecodefactory.org/mds/info-providers/

This information provider publishes data about the local software installed on a host. It advertises whether or not a piece of sofare is installed, it's path, and the version. 

  • Certificate authority info provider

http://www.thecodefactory.org/mds/info-providers/cert/

This information provider publishes information about the local certificate policy on each GRIS/GIIS host of a Globus based grid environment.

  • GridFTP Service information provider

This information provider that checks the GridFTP server on the local machine to make sure it can properly transfer files.

  • Teragrid Services Information Provider

http://www.thecodefactory.org/mds/info-providers/teragrid/

This information provider interfaces between a set of scripts developed for the teragrid and the MDS. JimBasney’s scripts  provide service information about what pieces of software and infrastrucure are currently running on a given system.
