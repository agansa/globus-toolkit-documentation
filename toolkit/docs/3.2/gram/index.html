GRAM Documentation

The Grid Resource Allocation Manager (GRAM) service provides a single interface for requesting and using remote system resources for the execution of "jobs". The most common use of GRAM is remote job submission and control. It is designed to provide a uniform, flexible interface to job scheduling systems.

GT3.2 contains two GRAM implementations: one based on a proprietary, pre-Web service protocol (Pre-WS GRAM) and the second built using Web service interfaces (WS GRAM).
