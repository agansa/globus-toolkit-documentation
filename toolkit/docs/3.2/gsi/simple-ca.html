The Globus Simple CA package provides a convenient method of setting up a certificate authority (CA) that is interoperable with the Globus Toolkit.  It is intended for users of small, test grid environments, or users that are not part of a larger grid.

UPDATE:  The Globus Simple CA is now included in the Globus Toolkit release (version 3.2 and up).  Please visit the Globus Toolkit website for further information.  The 3.2 Installation Guide provides updated documentation for the Simple CA install.

Users that have previous versions of the Globus Toolkit (3.0 and lower) can install the following bundle and read the Setup instructions following on this page.

  • globus_simple_ca-latest-src_bundle.tar.gz

Installation

Before proceeding with these setup instructions, you must have already installed the globus toolkit 2.X or greater. If you have already done so:

Set the GLOBUS_LOCATION environment variable to your globus toolkit 2.X installation. Then run the gpt-build command to install the globus_simple_ca bundle:

$GLOBUS_LOCATION/sbin/gpt-build \
globus_simple_ca_bundle-latest.tar.gz <flavor type>

(NOTE: Depending on how you installed globus, the gpt-build executable may be at your GPT_LOCATION.)

Setup

Please visit the Simple CA Setup Instructions page (part of the 3.2 setup instructions) for instructions on setting up the SimpleCA once the installation step is complete.
