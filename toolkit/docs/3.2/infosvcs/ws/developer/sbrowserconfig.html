Service Data Browser Configuration File (globus-sdb.conf)

Edit the Service Data Browser configuration file to set options for the Service Data Browser:

File Summary

Filename: globus-sdb.conf

Default location: <gt3-install-location>/etc (installed with GT 3.2)

The default contents are:

filterSubscriptions=false
registry=http://127.0.0.1:8080/ogsa/services/base/registry/Container
RegistryService
wrapTextInRawVisualizers=true

where:  
filterSubscriptions Controls the display of Notification Subscription instances in the Browser's “Select a service…” pull-down menu.  Each time a Grid service to which you are subscribed changes, it generates a Notification Subscription instance.  Setting this option to true turns on the display of Notification Subscription instances; setting it to false turns it off.
registry Specifies the URL of the Registry to which the Service Data Browser is connected at start up.  To specify a different registry, you can either change the value in the configuration file, or specify the registry (in the same format as in the configuration file) on the globus-sdb command line.  Specifying the registry on the command line overrides the value set in the configuration file.
wrapTextInRawVisualizers Controls text wrapping for the data displayed by the Raw Service Data and Raw WSDL Visualizers.  Setting this option to true turns on text wrapping; setting it to false turns it off.
