Globus Toolkit 3.2 Documentation

If you are new to the Globus Toolkit and are a developer interested in Grid services, we recommend reading the Toolkit Concepts.

Each component of the Toolkit contains the following manuals:
 
Key Concepts Provides a high level overview and required concepts for the component in layman's terms
Developer's Guide Provides reference information for application developers, including APIs, architecture, procedures for using the APIs and code samples.
System Administrator's Guide Provides information on procedures typically performed by system administrators, including installation, configuring, deploying, and testing the installation.
User's Guide For other users who do not typically have the high level knowledge of a developer or system administrator but must use the service (usually for scientific purposes). Includes basic procedures in simple terms, such as "How to Run a Client" and "How to Run a Personal Server".
