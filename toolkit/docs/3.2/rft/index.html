RFT Documentation

The Reliable File Transfer Service (RFT) is an OGSA based service that provides interfaces for controling and monitoring 3rd party file transfers using GridFTP servers. The client controlling the transfer is hosted inside of a grid service so it can be managed using the soft state model and queried using the ServiceData interfaces available to all grid services. It is essentially a reliable and recoverable version of the GT2 globus-url-copy tool and more.
