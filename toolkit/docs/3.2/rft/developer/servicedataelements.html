Service Data Elements for RFT

The following service data elements (SDEs) are available for the RFT service:

Version Specifies the version of RFT.
FileTransferProgress Specifies the percentage of file that is transferred.
FileTransferRestartMarker Specifies the last restart marker for a particular transfer.
FileTransferJobStatusElement Specifies the status of a particular transfer.
FileTransferStatusElement Specifies the status of all the transfers in the request.
GridFTPRestartMarkerElement Specifies the restart marker of the transfer.
GridFTPPerfMarkerElement Specifies the performance marker of the transfer.
