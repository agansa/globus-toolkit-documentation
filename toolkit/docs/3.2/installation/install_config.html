Configuring the Globus Toolkit 3.2

Follow all of the steps for Required Configuration and then look over the optional configurations for the components you want to use. The configurations described in this manual cover only basic settings that will allow you to get a service up and running. For more advanced options, please see the System Administrator's Guide for that technology.
