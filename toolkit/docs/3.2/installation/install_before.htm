Globus Toolkit 3.2: Installation Guide: Before You Begin

Before you start installing the Globus Toolkit 3.2, there are a few things you should consider.

Required Reading

If you are new to the Grid, make sure you understand these concepts:

  • Toolkit Concepts
  • OGSA Physiology

Core-only vs By-Component vs Full Installation

Consider which parts of the Globus Toolkit you need to install. If you only want to set up a basic test grid service, you can install the core alone. If you are only interested in the Core plus GRAM, then it is possible to install just those components. And of course, you can install the entire Globus Toolkit for more complicated ventures.

Users and user groups (GRAM)

If you are interested in the resource management tool, GRAM, consider your users and user groups. You will need users to install both the toolkit and the certificate authority as well as submit jobs.

user ID group ID ?whatever
root root super user
globus globus Globus Toolkit environment. For installation and execution of the Toolkit.
itso itso End user environment. For jobs execution on the Grid.

Note: It is important that the user ID under which you want to run GRAM is a member of the group ID that owns the launch_uhe_setuid program (see Configuring Resource Management.)
