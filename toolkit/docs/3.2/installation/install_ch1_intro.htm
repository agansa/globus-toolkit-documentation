This guide is the starting point for everyone who wants to install Globus Toolkit 3.2. It will take through a very basic installation that installs the Core and basic components: a security infrastructure (GSI), the index service (formerly referred to as MDS) and GridFTP. Once you have this basic setup, you'll be ready to go to the technology of your choice and find further installation and configuration information specific to that component.
