This guide is the starting point for everyone who wants to install Globus Toolkit 3.2. It will take you through a basic installation that installs Core and the following Base Services: a security infrastructure (GSI), the index service (formerly referred to as MDS) and GridFTP.

Once you have this basic setup, you'll be referred to the component of your choice for any further installation and configuration information specific to that component.
