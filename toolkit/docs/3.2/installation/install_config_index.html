Configuring GT3 Index Service - Basic

A basic configuration of the GT3 Index Service is installed with the Globus Toolkit 3.2. There are two main files that configure your index Service (follow these links for the parameters in each file that pertain to the GT3 Index Services):

  • Index Service Configuration file (index-service-config.xml)
  • Server Configuration file (server-config.wsdd)

Note: If you want to use the OGSA Service Browser GUI (is this the same as the GT3 Service Data Browser?), you will need the GUI control panels for the GT3 Index Service, which is configured in a different file.

Add the following lines to the client-gui-config.xml file (by default, you can find this in your GT3 installation directory): 

<panel portType="ServiceDataProviderExecutionPortType" class="org.globus.ogsa.gui.ProviderExecutionPortTypePanel"/>  
<panel portType="ServiceDataAggregatorPortType" class="org.globus.ogsa.gui.AggregatorPortTypePanel"/>

For more information, see the GT3 Index Service: System Administrator's Guide.
