Configuring WS Index Service - Basic

A basic configuration of the WS Index Service is installed with the Globus Toolkit 3.2.

There are two main files that configure your index Service (follow these links for the parameters in each file that pertain to the WS Index Services):

  • Index Service Configuration file (index-service-config.xml)
  • Server Configuration file (server-config.wsdd)

If you want to use the OGSA Service Browser GUI, see Enabling OGSA Service Browser GUI.

For more information, see the WS Index Service: System Administrator's Guide.
