Configuring RFT - Basic

Intro paragraph

Requirements

Before configuring RFT, you must have a host certificate (see Required Configuration.)

Basic Procedure
1

Create a postgres database for RFT to use.

There's an example .sql file included in the distribution you can use to create the database.
2

Open $GLOBUS_LOCATION/server-config.wsdd and look for the ReliableFileTransferFactoryService record.

Change the dbname and dbuser to match your database name and database user.

(It's probably a good idea to suggest the default dbname and dbuser so that this edit won't be required if they follow our instructions exactly)
3

Configure a GridFTP server.

See Configuring GridFTP for instructions.

For more information, see the RFT System Administrator's Guide.
