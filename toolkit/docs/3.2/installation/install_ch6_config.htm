Configuring the Globus Toolkit 3.2

The following is a rough flow of what you should do after installing the Globus Toolkit.

talk about Required vs optional configurations

Required configurations

  • Setting environment variables
  • Setting up the Grid Security Infrastructure (GSI)
  • Changing the ownership and access permission

Optional configurations

  • Setting up Resource Management/GRAM/MMJFS
  • Setting up Data Management (RLS/RFT/CAS?)
    • Setting up GridFTP and RFT
    • Setting up RLS
    • Setting up Security for Data Management (CAS?)
  • Setting up Index Services
  • Setting up pre-Web Services components
    • Setting up a GT GRAM service
    • Setting up a GT2 GIIS/GRIS
