Index Services

The Index Service component of the Globus Toolkit is available in two versions: the GT2 Index Service which is associated with the Globus Toolkit 2.x and the GT3 Index Service, associated with the Globus Toolkit 3.x and based on web services.

Choose one or the other:

  • GT3 Index Service (web services)
  • GT2 Index Service (pre-web services)
