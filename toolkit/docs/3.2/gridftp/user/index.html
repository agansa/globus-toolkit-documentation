Overview

Using GridFTP from a user's perspective is fairly easy.  The client provided with the Globus Toolkit is called globus-url-copy.  It is a scriptable, command line client.  It is relatively easy to use and gets its name from the fact that it can access multiple protocols that you can specify in a URL.
