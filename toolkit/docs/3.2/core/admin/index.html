Overview

This guide provides information on procedures typically performed by system administrators for the GT 3.2 Core, including installation, configuring, deploying, and testing the installation. If you have not installed the Globus Toolkit 3.2, you should start with the Installation Guide for basic information. This guide will include more advanced information specific to Core.
