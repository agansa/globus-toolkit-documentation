Note: Due to our recent [reorganization?] our mailing lists have been updated. There are now lists available for all official Globus Toolkit projects, as well as the overall GT lists and GT Release Manuals. There is also a new requirement to subscribe to the -user and -dev lists before posting messages. This is to prevent spammers from crashing the lists. The following are some general descriptions of the new lists:

  • -announce@globus.org: Information list that posts announcements relating to the project (not meant for posting).
  • -user@globus.org: Discussion list for end-users of the project.
  • -dev@globus.org: Discussion list for developers of the project.
  • -commit@globus.org: Information list that posts commits to the CVS repository for the project (not meant for posting).

Subscribing to a mailing list

To subscribe, send an email to majordomo@globus.org which contains the following in the body:

subscribe [list name]

end

Replace "[list name] " above with the prefix to the list address of your choice. For example, if you want to subscribe to the GT User Discussion list at gt-user@globus.org, use "gt-user" as the list name.

To verify you have been subscribed, you should receive an email from majordomo@globus.org with the subject, "Welcome to [list-name]". If you do not receive this email, you have not been subscribed!

Be sure the email address you are trying to subscribe matches the one majordomo sees. If they do not match, you will not be able to post to the mailing list, but only read it.

Before posting a question, you might find How to Ask Questions the Smart Way beneficial in getting your question answered. Please do not contact the authors of that document for help.

Un-subscribing from a mailing list

To un-subscribe, send an email to majordomo@globus.org which contains the following in the body: 

unsubscribe [list name]

Replace "[list name] " above with the prefix to the list address of your choice. For example, if you want to un-subscribe from the GT User Discussion list at gt-user@globus.org, use "gt-user" as the list name.
