Globus Toolkit 3.2 Alpha Download

DISCLAIMER: The Globus Toolkit 3.2 alpha release is, as the name indicates, of alpha quality. As such, there will be no official support for this release, stability may still be lacking and APIs as well as features are still subject to change. We advise that you do not use this release in a production setting.

Please read the Release Notes.

The "all services" bundle will install all 2.x and 3.x programs including core.
The "Core" packages only contains GT3 core, which is pure Java.

All services

  • GPT Source Installation Package

    For information about the contents, please see the Release Notes.

    To install, run './install-gt3' followed by './install-gt3-mmjfs'.
    Installation instructions can be found in the Admin Guide.

Core only
  • Core Source

    • This package is the OGSI portion of the release. It is pure Java, in source code or compiled. For install instructions, please see the User's Guide.

    Additional information is available in the GT3 Core white paper.

CVS Access

To check out code from our CVS:

  1. To login, use the following command: 
    cvs -d :pserver:anonymous@cvs.globus.org:/home/globdev/CVS/gridservices \
login
  2. At the password prompt, hit enter.

  3. Then check out code from the main trunk: 
    cvs -d :pserver:anonymous@cvs.globus.org:/home/globdev/CVS/gridservices \
co <module>
Replace <module> with one of the following modules:
  • ogsa
  • program_execution
  • data_management
  • information_services

To check out all available modules replace <module> with "all".

The 3.2 Alpha CVS tag is "globus_3_2a_pre5".
