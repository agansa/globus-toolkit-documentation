GSI-NCFTP

This release fixes the following bugs:

A bug in the procedure that copied the client TCP buffer caused errors during FTP "put" operation. The files after transmission would sometimes be larger than the originals, containing some trash appended to the original data. The error occurred at random and was more likely to happen with long files.

A separate buffer overflow bug in the protocol processing code was triggered by the latest GridFTP server update which fixed a minor security bug.  This problem appeared consistently on all platforms when using the most recent GridFTP server.

This package requires any one of the 2.0 source bundles to be previously built, or any one of the 2.0 binary sdk bundles to be previously installed.
