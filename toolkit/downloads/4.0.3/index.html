Globus Toolkit 4.0.3 Download

GT4.0 is an open source software toolkit used for building grids with services written in a combination of C and Java. The C components run on Unix platforms, including Linux. The java-only portions may be run on any platform with a Java SDK.

For an overview of how to customize your installation, see Globus Is Modular.

For installation instructions, please see the Globus Toolkit 4.0 Admin Guide or Globus Toolkit 4.0 Quickstart.

For information about the contents, please refer to the Release Notes or the full documentation.
