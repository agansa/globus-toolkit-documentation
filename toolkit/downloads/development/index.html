This page contains snapshots of Globus Toolkit code currently under development. The distributions on this page are known as "development releases" and are not covered by the same level of support and documentation as stable GT releases. For more information on stable and development releases, please read this page. The schedule for milestone development releases can be found here.

Some of these distributions have calls for testing associated with them - please see the Call for Testing page for more information.

Documentation for development releases can be found here.

  • November 3, 2006 16:30 PST -- MDS4 update package compatible with 4.0.3 releases.
    • The following package contains updates to the UsefulRP, Trigger Service, and WebMDS packages, and the addition of Nagios-based information providers. The UsefulRP update includes the new rpprovider component which is a modular framework for writing custom resource property information providers. Preliminary documentation for the rpprovider component can be found here. The Trigger Service update contains various usability enhancements, including; support for multiple TriggerRules per data aggregation, support for namespace mappings in the TriggerRule Xpath statement, invalidity time ranges for action script execution (scheduled downtime support, or "blackout intervals"), and additional options for configuring input to action scripts. Preliminary documentation for the new TriggerRuleType options can be found here. A sample Trigger Service registration file illustrating usage of the new options can be found here. The WebMDS update enables support for user-specified queries in WebMDS.
