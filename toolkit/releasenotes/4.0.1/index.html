Globus Toolkit 4.0.1 is an incremental release of the Globus Toolkit containing updates to the 4.0.0 release.

This page only lists the fixes since 4.0.0; for release notes specific to 4.0.0 go to:
http://www.globus.org/toolkit/releasenotes/4.0.0/.

Use the existing documentation for 4.0, which starts at http://www.globus.org/toolkit/docs/4.0/.
