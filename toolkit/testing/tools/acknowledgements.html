YourKit Java Profiler

   

The YourKit Java Profiler (YJP) is a commercial product from YourKit.com in St.Petersburg, Russia. Vladimir Kondratyev, a founder of the company, has generously donated YJP licenses to Globus developers in support of the Open Source status of the Toolkit.

This product covers the two most important tooling areas for Java development, namely Memory Debugging and Profiling.
