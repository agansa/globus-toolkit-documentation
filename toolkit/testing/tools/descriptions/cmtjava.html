Description
A commercial product for analysis of Code Complexity. It is produced by Verifysoft, and can be used to analyze Java and C++ code. It works on source code and is therefore platform independent. It provides many interesting metrics and should be evaluated by the DSL. It is a tool that would be used earlier in the development phase of modules as opposed to prior to releases.
