Description
A commercial package for Java and C/C++ applications produced by Rational and marketed by IBM. It runs on Windows, Linux and many Unix platforms. It has a very comprehensive suite of tools . As far as I know the Lab does not own a license to PurifyPlus. Schuler was using an evaluation copy. PurifyPlus should definately be evaluated for use with the Toolkit
