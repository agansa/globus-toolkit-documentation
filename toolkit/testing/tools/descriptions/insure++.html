Description
A commercial package for C/C++ applications produced by Parasoft. It runs on Windows, Linux and many Unix platforms. It has a comprehensive suite of tools for memory and resource checking and a coverage module. Insure++ requires a special compilation step to produce instrumented code. The Lab has an Insure++ site license and it is available on the servers. Joe Link, Mike Link, Raj, Joe Bester and others used this package extensively.
