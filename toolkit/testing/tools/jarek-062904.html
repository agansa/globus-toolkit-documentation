and ant in /home/dsl/javapkgs/apache-ant-1.6.1/ has clover installed.

dir.):

ant deployTest test generateCloverReport

p.p.s. More info on Clover if you are interested:

The website is here:

    http://www.thecortex.net/clover/index.html

This is a commercial product but they gave us a license to the Ant version
based on our status as an open source project. To set it up clover.jar is
copied to the ant directory and build.xml and build-tools.xml needed are
modified as described at:

    http://www.thecortex.net/clover/tutorial/tut0.html

Code is from the trunk:
   
    cvs co ogsa

Go to ./ogsa/impl/java

There are three steps to building the report: Instrument the code, run the
tests and create the report:

    ant with.cover all
    ant generateTestReport
    ant clover.html
 
(analysis was run on Linux it should run on Windows also)
