Java WS Core provides APIs and tools for developing Grid services and offers a run-time environment capable of hosting them. Java WS Core provides support for soft-state management, inspection, notification, discovery, and naming. It also provides advanced security infrastructure including different mechanism of transport and message level security.

Java WS Core in GT4

Java WS Core in GT4 is an implementation of the Web Services Resource Framework (WSRF) and the Web Service Notification (WSN) family of standards. It provides framework for building WS-Resources.
GT4 is the latest stable release and is highly recommended over GT3 as it offers a number of improvements in stability, scalability, performance, and usability.

Java WS Core in GT3

Java WS Core in GT3 is an implementation of the Open Grid Services Infrastructure (OGSI) specification. It provides framework for building Grid services.
