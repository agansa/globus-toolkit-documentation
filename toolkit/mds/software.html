MDS2 Viz Scripts

A set of flexible PHP scripts to show basic MDS2 information.

  • Download
Simple Ganglia MDS Information Provider

Connects to a ganglia monitoring daemon (gmond) or a ganglia Meta-daemon (gmetad).

  • Download from GTR
Teragrid Service Status Information Provider

Interfaces to a set of scripts to describe the status of some basic services. As part of the TeraGrid project, Jim Basney has created a series of scripts to describe the status of several resource services, which this IP interacts with.

  • More information (and Download)
Certificate Authority Information Provider

Advertises information about what user certificates are accepted at a site.

  • More information (and Download)
GridFTP Server Information Provider

Advertises the state of the GridFTP server installation, and works with any MDS2 installation.

  • More information (and Download)
Software Information Provider

Uses a small configuration file to report path and versioning information for software packages that are running on your machines.

  • More information (and Download)
