The Globus Project

The Globus Project is a collaborative effort which conducts research and development into the fundamental technologies behind Grid computing. Based at Argonne National Laboratory, the Information Sciences Institute, and the University of Chicago, it is organized around four main activities:

Research

The Globus Project studies problems in areas such as resource management, grid security, information services, and data management.

Core Software

The Globus Project works on building robust research software that enables technologies, including software core to Grid computing.

Building Grids

The Globus Project assists in planning and building large-scale testbeds, both for our own research and for production use by scientists and engineers.

Applications

The Globus Project develops large-scale, grid-enabled applications in collaboration with scientists and engineers.

Just as the World Wide Web has revolutionized the way we think about information, the Globus Project's goal is to achieve a similar result in computation. One can hardly imagine the types of applications we might construct if access to supercomputers and mass storage were as straightforward as access to the Web. That is the promise of Grid computing.

The Globus Toolkit

The Globus Toolkit is open source software being developed by the Globus Project and collaborators all over the world. Used for building grids, it exposes computing resources the way the Web exposes information. The toolkit contains several components which help it achieve its goal.

The Globus Toolkit has grown through an open-source strategy similar to the Linux kernel and is distinct from proprietary attempts at resource-sharing software. Its open-source approach and liberal license encourages broader, more rapid adoption and leads to greater technical innovation.

The upcoming release of the Globus Toolkit is based around OGSA and is working to:

  • Deliver an open source implementation of OGSI
  • Provide several, higher-level, OGSI-compliant services
  • Give developers the ability to create new OGSI-compliant services

The OGSI core is the foundation for the higher level grid services provided in the toolkit. It uses GSI to provide security and is the basis for all new grid services.
