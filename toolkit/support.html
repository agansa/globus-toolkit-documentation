Support for the Globus Toolkit is provided through documentation, software updates, training, mailing lists, GT4 Friends, and even Internet Relay Chat (IRC).

Documentation

If you have a question, please read the documentation first.

We are also pleased to announce the alpha release of the Globus Documentation Project (GDP), a community website that will serve to complement the official Globus documentation.

Software Updates

After you have read the appropriate documentation, please take a quick look at our advisories page to see if an update has been released which addresses your issue. The advisoris page contains software updates to our releases and is kept recent.

If you don't see an update and think you have found a bug in the toolkit, do a quick search in our Bugzilla for your bug. If it isn't already in the database, you can submit a new bug.

Before submitting a bug, you might find How to Report Bugs Effectively beneficial in expediting your bugs resolution.

Training

There are various training options available around the world, including presentations, workshops, commercial training and conferences:

  • GT Presentations
    • Globus Toolkit has published various tutorials and presentations; Globus Presentations includes links to Ian Fosters' recent talks as well as various overviews and powerpoint presentations.
  • Workshops
    • GRIDS Center Software Suite includes the Globus Toolkit and they offer various workshops and summer school options.
    • Commercial training
    • Univa, a company that develops and supports commercial Grid solutions based on Globus software, is introducing a series of courses and associated training materials, many of which are based on the tutorials developed by members of Univa's team while working with the Globus Project.
    • GridwiseTech offers a commercial 2-day training for GT4 covering the complete Globus Toolkit.
  • Conferences
    • GlobusWorld (GW) : For key players in Grid computing, GlobusWORLD has become the can't-miss event where leading technology producers and consumers debate and discuss real-world Grid technologies, tools and solutions. 
    • Super Computing Conference (SC): Participants in the Globus Alliance present a wide range of tutorials, technical papers, and posters at this premier international conference on high performance computing, networking and storage.
    • Global Grid Forum (GGF) is a community-initiated forum of thousands of individuals from industry and research leading the global standardization effort for grid computing.

Subscribing to a mailing list

To subscribe, send an email to majordomo@globus.org which contains the following in the body:

subscribe [list name]

end

Replace "[list name] " above with the prefix to the list address of your choice. For example, if you want to subscribe to the User Discussion list at discuss@globus.org, use "discuss" as the list name.

To verify you have been subscribed, you should receive an email from majordomo@globus.org with the subject, "Welcome to [list-name]". If you do not receive this email, you have not been subscribed!

Be sure the email address you are trying to subscribe matches the one majordomo sees. If they do not match, you will not be able to post to the mailing list, but only read it.

Before posting a question, you might find How to Ask Questions the Smart Way beneficial in getting your question answered. Please do not contact the authors of that document for help.

Un-subscribing from a mailing list

To un-subscribe, send an email to majordomo@globus.org which contains the following in the body: 

unsubscribe [list name]

Replace "[list name] " above with the prefix to the list address of your choice. For example, if you want to un-subscribe from the User Discussion list at discuss@globus.org, use "discuss" as the list name.

GT4 Friends

To participate in discussions regarding GT4 development activities such as documentation and testing, please subscribe to the gt4-friends@globus.org mailing list. More information on the GT4 Friends community group can be found here.

Internet Relay Chat (IRC)

For real-time discussion about the Globus Toolkit, you can log onto Freenode and join #globus. Please note that this forum is logged for support purposes. Recommended IRC clients for Windows are: mIRC and XChat and for Linux: XChat and irssi
