Status

jGlobus 2 is considered to be in beta and is not yet intended for production use. jGlobus is currently undergoing a security review. We encourage developers to download jGlobus and give early feedback. The security review is expected to be completed in late Spring or early Summer 2011. Upon completion of the security review, a production release of jGlobus will be announced here and on the jGlobus Announce mail list.

Source Downloads

Git

JGlobus source code is hosted in Github. To get JGlobus source code git has to be installed on your local machine. From a command prompt run the following command: 

    git clone http://github.com/jglobus

MyProxy

JGlobus MyProxy source code is hosted at SourceForge. To get JGlobus MyProxy source code, run the following command: 

    cvs -d:pserver:anonymous@cilogon.cvs.sourceforge.net:/cvsroot/cilogon co myproxy-jglobus
