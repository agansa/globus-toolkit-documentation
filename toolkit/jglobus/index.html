jGlobus

jGlobus is a collection of Java client libraries for Globus® Toolkit security, GRAM, and GridFTP. The primary changes in jGlobus 2 are the use of modern Java native SSL and modern Java packaging. This release was validated and tested against Globus® Toolit 5.0.3, but should also work with other subsequent 5.0.x and 5.2.x releases.

jGlobus 2 is in beta testing. We encourage developers to download jGlobus, try it and give feedback. The beta test period will end once sufficient user testing has completed.
