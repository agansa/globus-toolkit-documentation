jGlobus

jGlobus Project

jGlobus is a collection of Java client libraries for the Globus® Toolkit. jGlobus was recently upgraded to use Java native SSL. Validated and tested against Globus® Toolit 5.0.3, jGlobus is now at 2.0! See the jGlobus Download page for more information.
