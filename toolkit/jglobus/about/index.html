About jGlobus

The jGlobus Project

The jGlobus Project is a fork of the Commodity Grid (CoG) JGlobus Project. Its development is now led by the Globus Project and jGlobus efforts are no longer published under CoG or the Java CoG Kit. The motivation was to closely align the jGlobus library with the set of tools and services that are strictly contained within the Globus Toolkit. The CoG-JGlobus 1.8 branch was used as the basis for development.

The jGlobus Library

jGlobus is a collection of Java client libraries for the Globus® Toolkit. jGlobus was recently upgraded to use Java native SSL. Validated and tested against Globus® Toolit 5.0.3, jGlobus is now at 2! See the jGlobus Download page for more information.
