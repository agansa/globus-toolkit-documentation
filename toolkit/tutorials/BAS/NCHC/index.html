National Center for High-performance Computing, Taiwan

This two day event provides an in-depth introduction to the latest version of the Globus Toolkit (GT4). The Globus Toolkit is the flagship product of the Globus Alliance, which produces open source middleware used in building grids worldwide. The tutorials are geared toward developers who want to learn about the newest work of the Globus Alliance and how to apply fundamental concepts in Grid computing. Conducted by Ian Foster, Charles Bacon and Lisa Childers, this tutorial provides a solid introduction to this important technology.

November 9: Introduction to Globus Toolkit 4 (GT4)

The tutorial begins with a half day of presentations providing a comprehensive introduction to GT4. Topics covered include execution management services, data movement and management services, security services, and core runtime infrastructure.

November 10: How to Build a Service Using GT4

The second day teaches developers how to build a Java Service that makes use of GT4 mechanisms for state management, security, registry and related topics. The tutorial is organized as a series of hands-on exercises in which students add increasing functionality to a skeletal service implementation. Fundamental patterns and interactions of Grid computing are highlighted.
