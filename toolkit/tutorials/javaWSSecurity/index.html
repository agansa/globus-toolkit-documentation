Brief description

This tutorial provides an introduction to web services security and hands-on training on using various features of the Globus Toolkit to secure a Java web service. This tutorial also covers using the Delegation Service to delegate rights and discusses tools for access policy management in web services.

Pre-requisites

  • Basic knowledge of Java, XML
  • General understanding of Web Services/WSRF
  • Familiarity with GT Java WS Core. Refer to Build A Service Tutorial for introductory material.

Required Software

  • JDK 1.4.*
  • Apache Ant 1.6.2 or higher
  • Machines with static IP address and configured for correct resolution of hostname to IP Address.

Tutorial Material

  • Tutorial Document: Provides a complete walk-through of the tutorial, including installation steps.
  • Security Tutorial Presentations: Slides covering overview of concepts, GT 4 specific information and tutorial exercises.
